# 樱花特效配置指南

## 配置文件位置

`src/config/sakuraConfig.ts`

## 基本配置

```typescript
export const sakuraConfig: SakuraConfig = {
    // 是否启用樱花特效
    enable: true,

    // 樱花数量
    sakuraNum: 21,

    // 樱花越界限制次数，-1为无限循环
    limitTimes: -1,
}
```

## 启用/禁用樱花特效

```typescript
// 启用
enable: true,

// 禁用
enable: false,
```

## 樱花数量配置

```typescript
// 樱花数量
sakuraNum: 21,  // 数量越多，性能消耗越大
```

## 樱花尺寸配置

```typescript
size: {
    // 樱花最小尺寸倍数
    min: 0.5,
    // 樱花最大尺寸倍数
    max: 1.1,
},
```

## 樱花透明度配置

```typescript
opacity: {
    // 樱花最小不透明度
    min: 0.3,
    // 樱花最大不透明度
    max: 0.9,
},
```

## 樱花速度配置

```typescript
speed: {
    // 水平移动
    horizontal: {
        // 水平移动速度最小值
        min: -1.7,
        // 水平移动速度最大值
        max: -1.2,
    },
    // 垂直移动
    vertical: {
        // 垂直移动速度最小值
        min: 1.5,
        // 垂直移动速度最大值
        max: 2.2,
    },
    // 旋转速度
    rotation: 0.03,
    // 消失速度，不应大于最小不透明度
    fadeSpeed: 0.03,
},
```

## 层级配置

```typescript
// 层级，确保樱花在合适的层级显示
zIndex: 100,
```

## 常见问题

### Q: 樱花特效不显示？

1. 检查 `enable` 是否设置为 `true`
2. 检查浏览器控制台是否有错误
3. 尝试刷新页面或清除缓存

### Q: 如何减少性能消耗？

1. 减少 `sakuraNum` 数量
2. 增大樱花尺寸（尺寸大，数量可以减少）

### Q: 樱花挡住内容怎么办？

调整 `zIndex` 值，确保樱花在内容下方：
```typescript
zIndex: 100,  // 值越小，层级越低
```

### Q: 如何让樱花只飘落有限次数？

修改 `limitTimes`：
```typescript
limitTimes: 5,  // 飘落5次后停止
```
