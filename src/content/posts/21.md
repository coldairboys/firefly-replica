# B站视频嵌入指南

## 基本概念

在博客文章中嵌入 B 站视频可以丰富内容形式，提升用户体验。本指南将详细介绍如何在 Firefly（流萤）Astro 博客中嵌入 B 站视频及其他视频平台的内容。

## 1. 直接嵌入 B 站视频

### 使用 iframe 嵌入（推荐）

```markdown
<iframe 
  src="//player.bilibili.com/player.html?aid=12345678&cid=123456789&page=1" 
  scrolling="no" 
  border="0" 
  frameborder="no" 
  framespacing="0" 
  allowfullscreen="true"
  style="width: 100%; height: 400px;"
>
</iframe>
```

### 参数说明

- `aid` - 视频的 AV 号（如 AV12345678）
- `cid` - 视频的 CID（可从视频页面 URL 中获取）
- `page` - 视频分 P 数（默认为 1）

### 获取嵌入代码的方法

1. 打开 B 站视频页面
2. 点击视频下方的「分享」按钮
3. 选择「嵌入代码」
4. 复制生成的 iframe 代码

## 2. 响应式视频嵌入

为了在不同设备上显示良好，建议使用响应式嵌入：

```markdown
<div class="video-container">
  <iframe 
    src="//player.bilibili.com/player.html?aid=12345678&cid=123456789&page=1" 
    scrolling="no" 
    border="0" 
    frameborder="no" 
    framespacing="0" 
    allowfullscreen="true"
  >
  </iframe>
</div>

<style>
.video-container {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 比例 */
  height: 0;
  overflow: hidden;
}
.video-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
</style>
```

## 4. 视频嵌入最佳实践

### 性能优化

1. **懒加载视频**
   - 使用 `loading="lazy"` 属性
   - 或者使用第三方懒加载库

2. **视频尺寸控制**
   - 合理设置视频宽度和高度
   - 使用响应式布局

3. **预加载策略**
   - 仅在用户滚动到视频区域时加载
   - 避免自动播放

### 用户体验

1. **视频控制**
   - 启用全屏功能
   - 保留播放/暂停控制

2. **视觉设计**
   - 添加视频标题和描述
   - 保持与博客风格一致

3. **可访问性**
   - 添加视频的文字描述
   - 确保键盘导航支持

## 5. 常见问题

### 视频不显示

**原因：**
- 网络连接问题
- iframe 配置错误
- 视频已被删除或设置为私密

**解决方案：**
- 检查网络连接
- 验证嵌入代码
- 确认视频可公开访问

### 视频加载缓慢

**原因：**
- 视频分辨率过高
- 网络带宽不足
- 同时加载多个视频

**解决方案：**
- 使用懒加载
- 限制同一页面的视频数量
- 考虑使用视频缩略图替代

### 响应式显示问题

**原因：**
- 缺少响应式样式
- 固定尺寸设置

**解决方案：**
- 使用响应式容器
- 确保样式正确应用
