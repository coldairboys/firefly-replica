# B站视频嵌入指南

在博客文章中嵌入 B 站视频可以丰富内容形式，提升用户体验。本指南将详细介绍如何在 Firefly（流萤）Astro 博客中嵌入 B 站视频。

## 1. 获取视频嵌入代码

### 方法一：B站分享按钮获取

1. 打开 B 站视频页面
2. 点击视频下方的「分享」按钮
3. 选择「嵌入代码」
4. 复制生成的 iframe 代码

### 方法二：手动构造嵌入URL

```
//player.bilibili.com/player.html?bvid=视频BV号&page=1
```

## 2. 响应式视频嵌入

为了在不同设备上显示良好，建议使用响应式嵌入：

```html
<style>
.video-container {
  position: relative;
  padding-bottom: 56.25%;
  height: 0;
  overflow: hidden;
}
.video-container iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
</style>

<div class="video-container">
  <iframe
    src="//player.bilibili.com/player.html?bvid=BV1EZ4y1o74Z&page=1&high_quality=1&autoplay=false"
    scrolling="no"
    border="0"
    frameborder="no"
    framespacing="0"
    allowfullscreen="true"
  >
  </iframe>
</div>
```

## 3. 常用参数说明

| 参数 | 说明 | 示例值 |
|------|------|--------|
| `bvid` | 视频 BV 号 | `BV1EZ4y1o74Z` |
| `aid` | 视频 AV 号 | `12345678` |
| `page` | 视频分 P 数 | `1` |
| `high_quality` | 高质量播放 | `1` |
| `autoplay` | 自动播放 | `true` / `false` |

## 5. 常见问题

**视频不显示？**
- 检查网络连接
- 确认视频可公开访问
- 验证嵌入代码正确

**视频加载慢？**
- 使用懒加载
- 限制同一页面视频数量
- 考虑使用视频封面图替代
