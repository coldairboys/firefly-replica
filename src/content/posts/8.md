# 友链页面配置指南

## 配置文件位置

- 友链配置：`src/config/friendsConfig.ts`
- 页面配置：`src/config/friendsConfig.ts` 中的 `friendsPageConfig`
- 自定义内容：`src/content/spec/friends.mdx`

## 开启友链页面

在 `src/config/siteConfig.ts` 中开启：

```typescript
pages: {
    friends: true,  // 开启友链页面
},
```

开启后访问 `/friends/` 即可看到友链页面。

## 友链页面配置

```typescript
export const friendsPageConfig: FriendsPageConfig = {
    // 页面标题，如果留空则使用 i18n 中的翻译
    title: "",

    // 页面描述文本，如果留空则使用 i18n 中的翻译
    description: "",

    // 是否显示底部自定义内容（friends.mdx 中的内容）
    showCustomContent: true,

    // 是否显示评论区，需要先在commentConfig.ts启用评论系统
    showComment: true,

    // 是否开启随机排序配置，如果开启，就会忽略权重，构建时进行一次随机排序
    randomizeSort: false,
},
```

## 添加友链

编辑 `src/config/friendsConfig.ts` 中的 `friendsConfig` 数组：

```typescript
export const friendsConfig: FriendLink[] = [
    {
        title: "网站名称",
        imgurl: "https://example.com/avatar.jpg",
        desc: "网站描述",
        siteurl: "https://example.com",
        tags: ["Blog"],
        weight: 10,      // 权重，数字越大排序越靠前
        enabled: true,   // 是否启用
    },
    // 添加更多友链...
]
```

## 友链参数说明

| 参数 | 说明 | 示例 |
|---|---|---|
| `title` | 友链网站名称 | "我的博客" |
| `imgurl` | 友链头像图片 URL | "https://..." |
| `desc` | 网站描述 | "一个技术博客" |
| `siteurl` | 网站地址 | "https://..." |
| `tags` | 标签分类 | ["Blog", "Tech"] |
| `weight` | 排序权重 | 10 |
| `enabled` | 是否启用 | true |

## 权重排序

友链按 `weight` 值从大到小排序：

```typescript
weight: 10,  // 值越大越靠前
weight: 5,   // 值越小越靠后
```

## 随机排序

开启随机排序后，每次构建网站时友链顺序都会变化：

```typescript
randomizeSort: true,
```

## 自定义页面内容

在 `src/content/spec/friends.mdx` 中添加自定义内容，显示在友链页面底部：

## 评论区开关

```typescript
// 显示评论区
showComment: true,

// 隐藏评论区
showComment: false,
```

注意：需要先在 `commentConfig.ts` 中启用评论系统。

## 常见问题

### Q: 友链页面不显示？

1. 检查 `siteConfig.ts` 中 `pages.friends` 是否为 `true`
2. 检查友链配置中 `enabled` 是否为 `true`

### Q: 如何让友链不显示在某篇文章？

友链是页面级别的，不区分文章。

### Q: 友链图片加载失败？

检查 `imgurl` 是否为有效的图片 URL，确保图片可公开访问。

### Q: 如何申请友链？

在 `friends.mdx` 中添加申请说明，或者开启评论区让访客留言申请。

### Q: 友链排序不起作用？

检查 `randomizeSort` 是否为 `true`，如果开启则忽略权重排序。
