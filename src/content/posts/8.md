# 关于页、留言板与赞助页配置指南

这三个页面都很适合新手尽早配好，因为它们会直接影响访客对你博客的第一印象。

这篇会带你搞懂：

- 关于页改哪里
- 留言板改哪里
- 赞助页怎么开、怎么关、怎么放收款码

## 一眼看懂：这 3 个页面分别是什么

### 关于页

作用：

- 介绍你是谁
- 告诉别人这个站写什么
- 放你的定位、兴趣、折腾方向

### 留言板

作用：

- 让访客直接留言
- 适合打招呼、交流、提建议

### 赞助页

作用：

- 展示赞助方式
- 放收款码或赞助链接
- 可在文章页底部显示“赞助”按钮

## 关于页怎么改

### 文件位置

```text
src/content/spec/about.md
```

### 你现在这个项目里的关于页内容

就是直接从这个文件读取的，所以你想改文字，只要改这里就行。

### 一个最简单的示例

```markdown
# 关于本站

这里是 Coldairboy 的学习博客。

这个站主要记录：

- 学习笔记
- 博客折腾记录
- AI 工具使用心得
- 遇到问题后的排查过程
```

### 改完后访问哪里看

```text
/about/
```

## 留言板怎么改

### 文件位置

```text
src/content/spec/guestbook.md
```

### 页面开关位置

```text
src/config/siteConfig.ts
```

你要确保这里是开启状态：

```typescript
pages: {
  guestbook: true,
}
```

### 留言板内容怎么写

留言板页面上方的说明文字，就来自这个 Markdown 文件。

你可以写得更像自己，比如：

```markdown
---
title: "留言板"
description: "欢迎来这里打个招呼"
---

- 欢迎留言、交流、串门
- 如果你也在折腾博客、AI 工具、NAS、Docker，特别欢迎交流
- 有推荐的教程、项目或者踩坑经验，也可以直接分享
```

### 留言板评论从哪里来

留言板底部的评论区，走的是你全站当前启用的评论系统。

也就是说：

- 你如果启用了 Giscus
- 留言板下方就会显示 Giscus 评论区

### 留言板访问地址

```text
/guestbook/
```

## 赞助页怎么改

### 相关文件

主要看两个地方：

```text
src/config/siteConfig.ts
src/config/sponsorConfig.ts
```

### 第一步：确认赞助页开关开启

在 `src/config/siteConfig.ts` 里：

```typescript
pages: {
  sponsor: true,
}
```

### 第二步：改赞助页内容

在 `src/config/sponsorConfig.ts` 里可以改：

- 页面标题
- 页面描述
- 赞助用途说明
- 赞助方式
- 收款码图片
- 是否显示赞助者列表
- 是否在文章底部显示赞助按钮

### 最常改的配置

```typescript
export const sponsorConfig = {
  usage: "你的赞助将用于服务器续费、内容创作和功能维护。",
  showSponsorsList: true,
  showComment: true,
  showButtonInPost: true,
}
```

### 赞助方式怎么配

示例：

```typescript
methods: [
  {
    name: "微信",
    icon: "fa7-brands:weixin",
    qrCode: "/assets/images/sponsor/wechat.png",
    description: "使用微信扫码赞助",
    enabled: true,
  },
]
```

### 收款码应该放哪里

放在：

```text
public/assets/images/sponsor/
```

然后在配置里写对应路径，比如：

```text
/assets/images/sponsor/wechat.png
```

## 文章页底部的赞助按钮怎么控制

在 `src/config/sponsorConfig.ts` 里：

```typescript
showButtonInPost: true
```

如果你不想在每篇文章底部都显示赞助按钮，就改成：

```typescript
showButtonInPost: false
```

## 这三个页面最推荐的处理方式

如果你是个人博客，建议这样配：

- 关于页：认真写，像个人介绍页
- 留言板：打开，方便互动
- 赞助页：看你自己需求，没准备好就先关

## 常见问题

### Q: 我改了关于页内容，为什么页面没变化？

先检查：

1. 你改的是不是 `src/content/spec/about.md`
2. 本地页面有没有刷新
3. 你是不是改到了别的目录

### Q: 留言板为什么没有评论区？

通常是因为：

1. 评论系统没启用
2. 评论配置没填完整
3. 页面还没刷新

### Q: 赞助页打开是空的？

一般是因为：

1. 页面开关没开
2. `methods` 里都设置成了 `enabled: false`
3. 收款码路径写错了

## 最后总结

这三个页面其实很好理解：

- 关于页负责“介绍你是谁”
- 留言板负责“让别人能跟你聊”
- 赞助页负责“让愿意支持你的人知道怎么支持”

对于个人博客来说，它们都属于很值得早点配好的基础页面。
