# 文章加密、版权协议与分享海报指南

这篇讲的是文章页里比较实用的“增强功能”：

- 想让某篇文章加密码才能看
- 想给文章加版权协议
- 想启用分享海报按钮

这些都不是必须，但学会之后会明显更像一个完整博客。

## 一、密码文章怎么做

这个项目支持文章加密，核心字段有两个：

```yaml
password: "123456"
passwordHint: "密码提示"
```

### 直接写法示例

```markdown
---
title: 这是一篇加密文章
published: 2026-04-22
description: 只有输入密码后才能查看
tags: [学习笔记]
category: 博客搭建
author: Coldairboy
image: ./images/firefly1.avif
password: "coldairboy"
passwordHint: "和博客作者名字有关"
---
```

### 开启后会发生什么

- 文章列表里仍然能看到标题
- 进入文章页后，会先显示输入密码界面
- 输入正确密码后，正文才会显示

### 不想加密时怎么办

直接删掉：

```yaml
password:
passwordHint:
```

或者不要写这两个字段。

## 二、版权协议怎么配

这个项目支持文章底部显示版权信息。

### 默认版权配置文件

```text
src/config/licenseConfig.ts
```

你现在这个项目里默认是：

```typescript
export const licenseConfig = {
  enable: true,
  name: "CC BY-NC-SA 4.0",
  url: "https://creativecommons.org/licenses/by-nc-sa/4.0/",
}
```

### 单篇文章自定义版权

如果你想给某篇文章单独设置，可以在文章头信息里写：

```yaml
licenseName: "CC BY-NC-SA 4.0"
licenseUrl: "https://creativecommons.org/licenses/by-nc-sa/4.0/"
```

### 如果想关闭版权区块

全站关闭：

```typescript
enable: false
```

## 三、分享海报是什么

分享海报就是文章页里的一个按钮，点开后会生成一张适合分享的海报图。

海报里通常会带上：

- 标题
- 描述
- 日期
- 作者
- 二维码

## 分享海报开关在哪

在：

```text
src/config/siteConfig.ts
```

你要看这个字段：

```typescript
sharePoster: true
```

### 如果想关闭

```typescript
sharePoster: false
```

## 分享海报会用到哪些内容

生成海报时，通常会读取文章里的这些信息：

- `title`
- `description`
- `published`
- `image`
- `author`

所以如果你想让海报更好看，建议每篇文章尽量把这些都写完整。

## 一份适合新手的完整示例

```markdown
---
title: 我的隐藏教程
published: 2026-04-22
description: 这是一篇只有输入密码后才能查看的文章
tags: [学习笔记]
category: 博客搭建
author: Coldairboy
image: ./images/firefly1.avif
password: "coldairboy"
passwordHint: "和站长名字有关"
licenseName: "CC BY-NC-SA 4.0"
licenseUrl: "https://creativecommons.org/licenses/by-nc-sa/4.0/"
---
```

## 常见问题

### Q: 我写了 `password`，为什么还没加密？

先检查：

1. 是不是写在文章头信息里
2. 引号有没有写错
3. 本地页面有没有刷新

### Q: 分享海报按钮为什么不显示？

通常是因为：

1. `siteConfig.ts` 里的 `sharePoster` 被关了
2. 当前页面不是文章详情页
3. 本地缓存没刷新

### Q: 版权信息怎么换成自己的？

如果想全站统一改，就改 `licenseConfig.ts`。  
如果想只改某一篇，就在那篇文章头信息里单独写。

## 最后总结

这三个功能可以这样理解：

- `password`：控制“谁能看”
- `license`：说明“别人怎么转载”
- `sharePoster`：帮助“别人更方便分享”

它们都不是写文章的基础功能，但非常适合你在博客稳定后逐步加上。
