# 侧边栏、目录、站点统计与广告位配置指南

这个项目的侧边栏功能比很多主题都细，能控制：

- 左右侧栏怎么排
- 哪些组件显示
- 哪些组件只在文章页显示
- 站点统计开不开
- 广告位放不放

## 主要配置文件

```text
src/config/sidebarConfig.ts
```

这是最核心的文件。

## 侧边栏布局先看这几个参数

```typescript
export const sidebarLayoutConfig = {
  enable: true,
  position: "both",
  tabletSidebar: "left",
  showBothSidebarsOnPostPage: true,
}
```

### 它们分别是什么意思

- `enable`：是否启用侧边栏
- `position`：左侧、右侧还是双侧
- `tabletSidebar`：平板显示哪一侧
- `showBothSidebarsOnPostPage`：文章页要不要双侧栏

## 左右侧栏里的组件怎么控制

文件里会看到类似这种结构：

```typescript
leftComponents: [
  {
    type: "profile",
    enable: true,
    position: "top",
    showOnPostPage: true,
  },
]
```

你只要先理解 4 个字段就够了：

- `type`：组件类型
- `enable`：开关
- `position`：`top` 或 `sticky`
- `showOnPostPage`：文章页显不显示

## 文章目录怎么开

文章目录组件类型是：

```typescript
type: "sidebarToc"
```

示例：

```typescript
{
  type: "sidebarToc",
  enable: true,
  position: "sticky",
  showOnPostPage: true,
  showOnNonPostPage: false,
}
```

这表示：

- 只在文章详情页显示
- 跟随滚动吸附

## 站点统计组件怎么开

组件类型是：

```typescript
type: "stats"
```

示例：

```typescript
{
  type: "stats",
  enable: true,
  position: "top",
  showOnPostPage: true,
}
```

它一般会显示：

- 文章数量
- 最近更新
- 站点运行时间

## 广告位怎么开

广告组件本身是在侧边栏里控制开关，
具体广告内容在另一个文件里写：

```text
src/config/adConfig.ts
```

### 第一步：在 `sidebarConfig.ts` 里启用广告组件

```typescript
{
  type: "advertisement",
  enable: true,
  position: "sticky",
  showOnPostPage: true,
  configId: "ad1",
}
```

### 第二步：在 `adConfig.ts` 里写广告内容

比如纯图片广告：

```typescript
export const adConfig1 = {
  image: {
    src: "assets/images/cover.avif",
    alt: "广告横幅",
    link: "#",
    external: true,
  },
  closable: true,
  displayCount: -1,
}
```

或者带文字和按钮：

```typescript
export const adConfig2 = {
  title: "支持博主",
  content: "如果你觉得本站内容对你有帮助，欢迎支持创作。",
  link: {
    text: "支持一下",
    url: "about/",
    external: false,
  },
}
```

## `top` 和 `sticky` 有什么区别

这是新手很容易看懵的地方。

### `top`

适合：

- 个人资料
- 公告
- 站点统计

特点：

- 显示在侧栏上方
- 更像“固定在最前面的内容块”

### `sticky`

适合：

- 目录
- 分类
- 标签
- 广告

特点：

- 会随着页面滚动保持可见

## 我建议新手这样配

如果你现在刚开始用，先用这套最稳：

### 左侧栏

- 个人资料
- 音乐播放器
- 分类
- 标签

### 右侧栏

- 公告
- 站点统计
- 文章目录

这样信息会比较清楚，不容易太乱。

## 常见问题

### Q: 我开了目录，但文章页没有显示？

先检查：

1. `type` 写的是不是 `sidebarToc`
2. `enable` 是不是 `true`
3. `showOnPostPage` 是不是 `true`
4. 当前文章里有没有足够的标题层级

### Q: 广告位为什么没显示？

一般是因为：

1. 侧边栏里没启用广告组件
2. `configId` 写错了
3. `adConfig.ts` 里内容不完整

### Q: 站点统计为什么不对？

通常先检查：

1. 文章发布时间有没有乱填
2. `siteStartDate` 有没有写对
3. 页面是不是刷新成功了

## 最后总结

这个项目的侧边栏不是“固定死”的，而是可以自己搭。

你可以把它理解成：

- `sidebarConfig.ts` 决定“放什么、放哪边、什么时候显示”
- `adConfig.ts` 决定“广告具体长什么样”

先把布局逻辑搞明白，后面调整就会很轻松。
