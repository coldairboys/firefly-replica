# 公告配置指南

## 配置文件位置

`src/config/announcementConfig.ts`

## 基本配置

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "公告",
    content: "Hi，欢迎来到 Coldairboy'学习笔记。",
    closable: true,
    link: {
        enable: true,
        text: "了解更多",
        url: "/about/",
        external: false,
    },
}
```

## 配置参数说明

| 参数 | 说明 | 示例 |
|---|---|---|
| `title` | 公告标题 | "公告" |
| `content` | 公告内容 | "欢迎来到我的博客" |
| `closable` | 是否可关闭 | true |
| `link.enable` | 是否显示链接 | true/false |
| `link.text` | 链接文字 | "了解更多" |
| `link.url` | 链接地址 | "/about/" |
| `link.external` | 是否外链 | true/false |

## 配置示例

### 简单公告（无链接）

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "公告",
    content: "网站正在维护中，请稍后再访问。",
    closable: true,
    link: {
        enable: false,
    },
}
```

### 带链接的公告

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "新功能上线",
    content: "博客全新版本已上线！",
    closable: true,
    link: {
        enable: true,
        text: "查看更新日志",
        url: "/posts/changelog",
        external: false,
    },
}
```

### 外部链接公告

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "活动公告",
    content: "关注我的社交媒体获取最新动态！",
    closable: true,
    link: {
        enable: true,
        text: "前往 Twitter",
        url: "https://twitter.com/username",
        external: true,
    },
}
```

### 不可关闭的公告

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "重要通知",
    content: "请所有访客注意账户安全。",
    closable: false,  // 设置为 false，用户无法关闭
    link: {
        enable: false,
    },
}
```

### 完全禁用公告

如果想完全禁用公告功能，可以将内容留空：

```typescript
export const announcementConfig: AnnouncementConfig = {
    title: "",
    content: "",
    closable: true,
    link: {
        enable: false,
    },
}
```

## external 参数说明

| 值 | 说明 |
|---|---|
| `false` | 内部链接，不会在新标签打开 |
| `true` | 外部链接，会在新标签打开 |

## 常见问题

### Q: 公告不显示？

1. 检查 `content` 是否为空
2. 检查配置文件是否正确导入
3. 检查浏览器控制台是否有错误

### Q: 想只显示标题不显示内容？

```typescript
content: "",
```

### Q: 链接指向首页？

使用相对路径：
```typescript
url: "/",  // 指向首页
```

### Q: 公告样式可以修改吗？

公告样式由主题控制，暂不支持自定义样式修改。

### Q: 关闭后还能重新打开吗？

关闭后需要清除浏览器缓存或重启服务器才能重新显示（如果 `closable` 为 `true`，用户关闭后不会再次显示）。
